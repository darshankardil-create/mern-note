# MERN Notes Backend

A **Node.js + Express** backend for managing notes with **MongoDB**. Provides a REST API for creating, reading, updating, and deleting notes. Can be connected to any frontend.

---

## Features

- Create, read, update, and delete notes
- MongoDB database with Mongoose
- Structured MVC architecture (Models, Controllers, Routes)
- Ready for integration with React or other frontends
- Environment variables for secure configuration

---

## Tech Stack

- **Backend:** Node.js, Express.js
- **Database:** MongoDB, Mongoose
- **Environment Management:** dotenv
- **Others:** CORS
